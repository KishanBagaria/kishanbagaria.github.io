One Click Llama Button for DeviantArt

One Click Llama Button finds all deviant and group names in the page and adds a button next to them which lets you give them a Llama in just one click.

Installing One Click Llama Button

For Chrome/Firefox/Opera:

  1. Update your browser, if you aren't using the latest version.

  2. Install an extension in your browser which lets you use userscripts (One Click Llama Button is a userscript):
    Chrome: Tampermonkey
    Firefox: Greasemonkey (you have to restart Firefox after installing Greasemonkey)
    Opera: Violentmonkey
    Safari: Tampermonkey
    Android: Tampermonkey Dolphin

  3. Click One Click Llama Button. After you click this link, a page or dialog will appear where you'll get further instructions for installing it. If you aren't sure, just find and click anything that says Install on that page.

When you're done installing, open/refresh any DeviantArt page and start spreading the Llama love! Don't forget to share this with your friends so that more people can join the fun!

Upgrading on Safari

Earlier there was no userscript extension available for Safari so a separate One Click Llama Button extension was available for download. If you are using the separate extension, uninstall it and reinstall One Click Llama Button with the new instructions to get the latest version.
